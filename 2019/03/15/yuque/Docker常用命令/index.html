前言
记录一下 docker 常用命令方便查找
Docker可以认为是 vmware 或者 virtualbox
•   镜像可以认为是 xxx.iso。
•   容器可以认为是 virtualbox 运行 xxx.iso 后的系统。
国内镜像加速器：https://www.daocloud.io/mirror#accelerator-doc

从镜像仓库拉取镜像

命令： docker pull 仓库+镜像:版本号

推送本地镜像到镜像仓库

命令： docker push 仓库+镜像:版本号
